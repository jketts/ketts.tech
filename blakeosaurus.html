Using 123D catch - quick prototyping.

Coming soon! This project is still a work in progress..

This is an old write up of a little afternoon project I completed a year or so ago. It was kind of interesting so I'm going to give a quite write up to remember how I did everything and keep the relevant files.
So I made 2 things. The first was a Blakeosaurusrex.

  • First download 123D catch onto your phone or PC. I only installed it on my computer. So first take a load of photos. I went around my friend Blake twice taking photos. Once at eye level and once from slightly about him - this allows the software to work out when the top of his head looks like.
  • So I have my phone set up to upload all photos I take staight to dropbox and sync anytime wifi is connected. So by the time time Ive finished taking the photos they are already synced to my PC. Then you can easily use 123D catch software to generate the 3D model of the head.
  • I then use meshmiker to cut the two models together and save the final file as an .stl for the 3D printers. Always use a mouse when trying to use meshmixer - I always forget this is the case when using the program for the first time in a while and find my model impossible to control on a trackpad.
  • The second thing to make was something to use as a test for the kiln im building. For this I found two files on... The first is was a deer skull and the second is a bust of a mariener with a sweet beard. Using meshmixer again I simply clipped the horns o the rams skull and attatched them to the bust. This was then printed on one of the 3D printers.
  • The next step... Take a latex mould of of the 3D print so we can work towards metal casting!

So I've been building a kiln at home/at the Uno. Why? Well I've been making a few items on some of the 3D printers in the uni, and it would be really nice if i could make them out of something other that week plastic. There is a titanium printer and a milling machine.. but I thought that the easiest way to take an item from idea to physical form maybe to do the following:

  1. Make a 3D diesign in ...
  2. 3D print the design
  3. Make a latex cast of the design from the 3D print.
  4. Using this cast we then then make several wave models
  5. Using the wax model we then make plaster casts
  6. Then melt out the wax and fill the cast with molten metal
  7. Smash the plaster cast
